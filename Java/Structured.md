# 💻 Platform Independent কী?

Java এর সবচেয়ে বড় বৈশিষ্ট্য হলোঃ

> **Write Once, Run Anywhere**

অর্থাৎ Windows-এ লেখা Java Program Mac, Linux সহ অন্য প্ল্যাটফর্মেও চলতে পারে।

---

## Java Compilation Process

```text
Program.java
     ↓
 Compiler
     ↓
Program.class
     ↓
 JVM
     ↓
 Output
```

---

# ⚙️ JVM কী?

JVM = Java Virtual Machine

এটি Java Program চালাতে সাহায্য করে।
JVM এর ভিতরে Interpreter থাকে যা `.class` ফাইল রান করে।

---

# 🧠 Garbage Collection কী?

Java-তে অপ্রয়োজনীয় Memory স্বয়ংক্রিয়ভাবে Delete হয়ে যায়।

এটিকে বলা হয়ঃ

> Garbage Collection

এ কারণে Java-তে C++ এর মতো `delete` ব্যবহার করতে হয় না।
